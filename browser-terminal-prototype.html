Trying out the terminal

Click Terminal to open a session. Just like connecting over SSH, the terminal opens first and then asks for the root password inline — use hunter2 for this demo (anything else returns “Permission denied”). Once you're in, it behaves like a real shell. A few things worth trying:

  • Run package commandsapt update, apt-get update, or apt install nginx return realistic, streaming output.
  • Copy & pasteCopy a command from anywhere and paste with Ctrl+V or right-click; selecting and copying from the terminal works too.
  • Open linksRun wget https://wordpress.org/latest.tar.gz — printed URLs render as clickable links that open in a new tab.
  • Real shortcuts↑/↓ history, Tab to complete, Ctrl+L clear, Ctrl+C cancel, Ctrl+D log out.

Type help any time for the full command list.
