Existing customer journey · self-serve connect

Connect Nexos AI to your app manually

For apps Bluehost doesn’t auto-deploy (Sim, Flowise, GStack, BMAD…), add your Nexos key yourself. Connector apps (OpenClaw, Hermes Agent, n8n) can instead connect in one click from the panel.

Add your Nexos key — step by step

~5 minutes
nxs_live_••••••••••••4f9a

Your full key is on your Nexos AI page. Replace the placeholder below with it.

1

SSH into your VPS and open the app’s environment file

Connect over SSH using the IP and root password from your VPS service page. AI Agent apps installed from the one-click library keep their settings in an .env file inside the app folder (usually /opt/<app>/). Open it with a text editor like nano.

# connect to your server ssh root@<your-vps-ip> # go to the app directory and open its env file cd /opt/openclaw nano .env
2

Set the Nexos key as your provider key

The Nexos gateway is OpenAI-API compatible, so you point the app’s existing OpenAI settings at Nexos. Add (or update) these three lines in the .env file, then save and exit nano with Ctrl+O, Enter, Ctrl+X.

# paste your real key from the Nexos AI page NEXOS_API_KEY=nxs_live_xxxxxxxxxxxx4f9a OPENAI_API_KEY=nxs_live_xxxxxxxxxxxx4f9a OPENAI_API_BASE=https://gateway.nexos.ai/v1
3

Restart the app container so it picks up the new variable

Environment changes only apply on restart. If your app runs with Docker Compose, recreate it; for a single container, restart it by name. Then confirm it’s back up.

# docker compose setup cd /opt/openclaw docker compose down && docker compose up -d # or, single container docker restart openclaw # confirm it is running docker ps
4

Send a test prompt to confirm it works

Open your app in the browser and send any message — or test the gateway directly from the server with curl. A normal reply means the app is now drawing from your Nexos wallet. Your balance on the Nexos AI page will tick down as you use it.

curl https://gateway.nexos.ai/v1/chat/completions \ -H "Authorization: Bearer $NEXOS_API_KEY" \ -H "Content-Type: application/json" \ -d '{"model":"claude-sonnet-4","messages":[{"role":"user","content":"ping"}]}'

Disconnect from Nexos

This removes your Nexos API key from your app and requires you to add your own LLM provider key to keep the app working.

Your Nexos credit balance is unaffected — only this app’s wiring changes.

Rotate your API key

Rotate only if you think your key is leaked. On confirm:

  • A new key is generated immediately
  • The old key is revoked instantly
  • Your app will stop working until you reconnect it with the new key
After rotation you’ll reconnect your app yourself — one click from the panel, or by following the self-serve KB. Nothing re-wires automatically. Your wallet, balance, expiry date and usage history are not affected.
